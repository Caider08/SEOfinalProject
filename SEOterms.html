    SEO Glossary
    Term Acronym Definition Link
    Relevance When a person performs an online search, the search engine returns only those results that are relevant to the searcher's query. Relevance
    Popularity Search engines typically assume that the more popular a site, page, or document, the more valuable the information it contains must be. Popularity
    Click-Through-Rate CTR The percentage of users who click on a given domain afer entering a search query. Popularity
    Search Engine Results Page SERP The listing of web pages returned by a search engine after a search query is made. Popularity
    Organic Results The listing of web pages in the query results that are not paid advertisements. Popularity
    Paid Results Advertisements on search engines that appear in the query results...typically near the top(first few results) Popularity
    robots.txt Indicate whether certain user agents(web-crawling software) can or cannot crawl parts of a website. Robots.txt
    keyword stuffing Keyword stuffing is when someone attempts to manipulate their position in the search results by concentrating relevant keywords. Keyword Stuffing
    meta description tag Meta descriptions are HTML attributes that provide concise summaries of webpages. They commonly appear underneath the blue clickable links in a search engine results page (SERP) meta description tag
    keyword cannibalization Happens when a website's information architecture rely on a single keyword or phrase on multiple places of the website. meta description tag
    pinging Pinging is a way for you to electronically let search engines and other websites know that you have new or revised content on your website. pinging
    DMCA takedown DMCA When content is removed from a website at the request of the owner of the content or the owner of the copyright of the content. pinging
    Cloaking SEO Technique in which the content presented to the search engine spider is different from that presented to the user's browser. Cloaking
